Smart Flix Recommendation System

Overview:
Smart Flix Recommendation System is a Python-based application that recommends movies based on the user's favorite genre. The system provides movie suggestions from different categories and helps users discover movies according to their interests.

Objectives:
- To develop a simple movie recommendation system using Python.
- To understand the concept of recommendation systems.
- To provide personalized movie suggestions based on user preferences.

Features:
- Genre-Based Movie Recommendations
- Multiple Movie Suggestions
- Random Recommendation Selection
- User-Friendly Console Interface
- Supports Different Movie Categories
- Continuous Recommendation Option

Technology Used:
- Python
- Visual Studio Code

How to Run:
1. Open Terminal or Command Prompt.
2. Navigate to the project folder.
3. Run the Python files 
   python movie_recommendation.py
4. Enter your preferred movie genre.
5. View the recommended movies.

Output:
- Displays available movie genres.
- Recommends movies based on user choice.
- Allows users to get recommendations multiple times.
- Provides a simple interactive experience.

Learning Outcomes:
- Python Basics
- Dictionaries and Lists
- Loops and Conditional Statements
- Random Module Usage
- User Input Handling
