Learn Mate AI

Overview:
Learn Mate AI is a Python-based educational assistant that helps students learn technical topics, practice viva questions, take quizzes, explore career paths, and stay motivated through daily quotes.

Objectives:
- To develop a simple learning assistant using Python.
- To help students learn basic technology concepts.
- To provide quiz and viva practice.
- To guide students towards different IT career paths.

Features:
- Learn Topics (Python, AI, ML, Data Science, Cyber Security, Cloud Computing)
- Interactive Quiz Section
- Viva Practice Questions and Answers
- Career Guidance Roadmaps
- Daily Motivation Quotes
- User-Friendly Menu Driven Interface

Technology Used:
- Python
- Visual Studio Code
