Smart Tic Tac Toe Challenge

Overview:
Smart Tic Tac Toe Challenge is a Python-based console game where a player competes against the computer. The game includes winner detection, draw detection, input validation, and replay functionality to provide an engaging gaming experience.

Objectives:
- To develop an interactive Tic Tac Toe game using Python.
- To implement game logic and decision-making.
- To improve programming and problem-solving skills.

Features:
- Human vs Computer Gameplay
- Random AI Move Generation
- Winner Detection System
- Draw Match Detection
- Input Validation
- Replay Option
- Simple Console Interface

Technology Used:
- Python 3
- Visual Studio Code

How to Run:
1. Open the project folder.
2. Open Terminal or Command Prompt.
3. Run the Python file:
   python tic_tac_toe.py
4. Enter a position between 1 and 9 to make your move.
5. Continue playing until a winner is declared or the match ends in a draw.

Output:
- Displays the Tic Tac Toe board.
- Allows the user to play against the computer.
- Announces the winner or draw result.
- Provides an option to play again.

Learning Outcomes:
- Functions in Python
- Lists and Indexing
- Conditional Statements
- Loops
- Exception Handling
- Game Logic Development
